# Sunrin Today

Sunrin Internet High School Lunch Service API

<br>

## Installation

```sh
git clone https://github.com/sunrin-today/api
```

## Build

```sh
docker build . -t ghcr.io/sunrin-today/api
```

## Start

```sh
docker run ghcr.io/sunrin-today/api
```

## Contributing
